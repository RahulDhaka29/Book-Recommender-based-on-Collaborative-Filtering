About This Project

This Book Recommender System is a machine learning project designed to help users discover new books based on their interests. It uses two different recommendation approaches:

  • Popularity-Based Filtering: The homepage showcases the top 50 books from the dataset, ranked by a combination of high average ratings and a significant number of votes. This is great for new users who want to see what's generally popular.
  • Collaborative Filtering: The "Recommend" page uses a more personalized approach. When you enter a book you like, the system finds users who also liked that book and then recommends other books that those similar users enjoyed. This is based on the idea that people with similar tastes will like similar things.

Dataset Used

This project is powered by the Book-Crossings dataset from Kaggle. It contains 1.1 million ratings of 270,000 books by 92,000 users.
